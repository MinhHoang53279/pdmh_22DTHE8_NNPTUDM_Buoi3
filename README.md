# Dashboard Sản phẩm - NNPTUD Buổi 3

**Họ và tên:** Phạm Đồng Minh Hoàng  
**MSSV:** 2280601059

## Mô tả

Dashboard hiển thị sản phẩm sử dụng API: https://api.escuelajs.co/api/v1/products

## Tính năng

- Lấy và hiển thị danh sách sản phẩm từ API
- Tìm kiếm theo tên sản phẩm
- Sắp xếp theo giá (tăng/giảm) và tên (A-Z, Z-A)
- Phân trang với lựa chọn 5, 10, 20 mục mỗi trang
- Hiển thị hình ảnh sản phẩm

## Cấu trúc

- `dashboard.html` - Giao diện chính
- `styles.css` - Style
- `script.js` - Logic và xử lý dữ liệu

## Chạy thử

Mở file `dashboard.html` bằng trình duyệt hoặc dùng Live Server (VS Code).
